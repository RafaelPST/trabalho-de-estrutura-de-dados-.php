# Coleta de Links em PHP

Projeto da materia de estrutura de dados para praticar web scraping em PHP utilizando parsing de HTML e exportação de dados para CSV.

## Arquivos
- coleta_links_local.php → versão para execução local (usa requisição HTTP real)
- coleta_links_online.php → versão adaptada para ambientes online (HTML simulado)

## Como executar (local)
```bash
php coleta_links_local.php
